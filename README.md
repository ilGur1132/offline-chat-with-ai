Offline Chat with AI is an extension that enables you to chat with the built-in AI chatbot in your browser. 

In Chrome browser version 138+, the Gemini Nano is available as an offline AI engine. This extension lets you chat with this AI agent through a very user-friendly interface. Please note that when you first open the app, it needs to download training data. Read more about this built-in AI here: https://developer.chrome.com/docs/ai/built-in

The app interface has a toolbar on the left side with a few buttons (self-explanatory with tooltips). Moreover, the interface can be opened in the toolbar popup, sidebar, separate window, or a new tab. Please right-click on the toolbar button to choose one.
You can install this app as a PWA on your mobile device, moreover, it is available as a browser extension as-well-as a web application:
